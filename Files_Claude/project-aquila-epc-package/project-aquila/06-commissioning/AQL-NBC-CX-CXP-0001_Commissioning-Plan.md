## 1. Commissioning Levels (L1–L5)
| Level | Name | Tag colour | Scope |
|---|---|---|---|
| L1 | Factory Acceptance Test (FAT) | — | Verify equipment at factory vs spec before dispatch |
| L2 | Delivery & Pre-Installation | Yellow | Verify delivered, undamaged, complete |
| L3 | Pre-Functional Test (PFT) | Green | Verify correct installation before energisation |
| L4 | Functional Performance Test (SAT) | Blue | Individual systems under load & fault scenarios |
| L5 | Integrated Systems Test (IST) | White | Whole-facility integrated failure/black-building test |

## 4. IST Philosophy (L5)
Full-load bank testing with simulated utility failure, generator start, UPS ride-through, and
cooling ride-through (TES-01). Demonstrate concurrent maintainability (Tier III) by isolating
each A/B path in turn.
