## 1. Scope
Receipt, positioning, anchoring, terminating and pre-energisation checks of DH1 UPS system
following delivery inspection (MIR MGD-ATL-QA-MIR-0031, L2 Yellow Tag).

## 2. Sequence of Work
1. Confirm Yellow Tag (L2) acceptance and area readiness (raised floor, earth grid per DWG-1030).
2. Rig and position UPS-01…06 to layout DWG-1010/1011; anchor battery cabinets with **Zone II
   seismic kit** (per NCR-0037 rework — verify anchor torque).
3. Install STS and Maintenance Bypass Panels; land power cables per Cable Schedule
   MGD-ATL-EL-CBS-0004 (C-DH1-0101…0205); torque and record.
4. Bond clean earth per Site Instruction MGD-ATL-EL-SI-0044 (cable C-DH1-EARTH1).
5. Megger/insulation-resistance checks before energisation (record for L3).
6. Handover to commissioning for L3 pre-commissioning (MGD-NBC-CX-PCC-0012).

## 3. Hold Points (align to ITP-0009)
- H1 Seismic anchoring inspection (item 7) — Atlas QA witness.
- H2 Cable termination torque records complete before energisation.
- H3 Permit to Work + LOTO in force before any live work (MGD-ATL-HS-PTW-0231 / LOTO-0001).

## 4. Resources / Plant
Certified electricians (×6), rigging crew, torque tools (calibrated), IR tester, forklift.

## 5. HSE
Hot Work Permit for any cutting/grinding; LOTO on LV-SWGR-A1/B1 feeders during termination.
Toolbox talk (MGD-ATL-HS-TBT series) each shift.
