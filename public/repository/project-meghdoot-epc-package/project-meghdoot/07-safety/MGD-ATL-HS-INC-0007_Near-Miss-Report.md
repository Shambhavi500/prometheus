## Description
During rigging of UPS-03, a lifting sling showed fraying not identified at pre-use inspection.
Load was lowered and the lift stopped before any failure. No injury, no equipment damage.

## Immediate Action
Sling quarantined and destroyed. Lift resumed with certified replacement.

## Root Cause & Corrective Action
Pre-use inspection register not enforced for slings. Corrective: daily colour-code sling
inspection tags introduced; toolbox talk delivered (MGD-ATL-HS-TBT-0142). Referenced in
Lessons Learned MGD-ATL-PM-LL-0001.
